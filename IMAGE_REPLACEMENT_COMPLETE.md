## Next.js Image Component Migration (Optional)

For better performance, consider migrating from `<img>` tags to Next.js `<Image>` component:

```tsx
// Before
<img src="/images/afri-rise/african-business-meeting-1.jpg" alt="..." />

// After
import Image from 'next/image'
<Image 
  src="/images/afri-rise/african-business-meeting-1.jpg" 
  alt="..."
  width={1920}
  height={1080}
  priority
/>
```
